  An open-source, deterministic cost estimator and analyzer for the Walrus decentralized storage protocol. It allows developers and system designers to measure, visualize, and plan storage costs on Walrus with real, on-chain data. 

  Built to be **fast, transparent, and engineer-friendly**, WalCost helps you make informed decisions when designing storage systems.




---

## Features

- **Quick Estimate**: Instantly see WAL token and SUI gas costs for any blob size, storage type, and duration.
- **Analyzer Mode**: Evaluate system-level scenarios such as periodic uploads, retention policies, and break-even points.
- **Transparency**: Access raw measurement data to verify and audit cost calculations.
- **Deterministic**: All calculations come from real measurements, ensuring predictable and reliable estimates.
- **Open-Source**: Fully OSS-friendly; contributions are welcome.

---

## Installation

```bash
# Clone the repo
git clone https://github.com/amosehiguese/walcost.git
cd walcost

# Install dependencies
pnpm install

# Start development server
pnpm dev

```
